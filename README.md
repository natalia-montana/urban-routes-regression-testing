# Urban Routes: Regression Testing & UI Validation

## 📝 Project Description
This repository contains the quality assurance analysis and regression testing execution for the **Urban Routes** web application, a smart transportation and route-optimization platform. The main goal of this project was to validate core user flow interactions, ensure cross-browser consistency, and execute thorough UI component checks to safeguard the end-to-end user experience (CX).

## 🛠️ Technologies & Tools Used
* **Test Design Techniques:** Equivalence Partitioning, Boundary Value Analysis (BVA)
* **Documentation & Artifacts:** Google Sheets / Excel
* **Version Control:** GitHub

## 📋 Key Testing Activities Covered
* Verification of critical paths and button behaviors within the main interface.
* Execution of a full regression suite to ensure code stability after minor updates.
* Cross-browser UI validation for consistent layout rendering.
